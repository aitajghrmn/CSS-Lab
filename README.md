# CSS Experiments

A collection of advanced CSS effects, hovers, animations, abstract UI ideas, and tricky frontend experiments.

## Categories
- Hovers
- Buttons
- Cards
- Text Effects
- Loaders
- Backgrounds
- Borders
- Misc Experiments

## Goal
This repo is for creative, difficult, and visually interesting CSS snippets that are hard to find or hard to invent from scratch.

## Usage
Open any folder and run `index.html` in your browser.

## Contributions
You can add new effects by creating a new folder with:
- `index.html`
- `style.css`
- `preview.png`
